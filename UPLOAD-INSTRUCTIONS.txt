VANGUARD FLEET AWARDS PAGE REPLACEMENT

This file is named awards.html so it directly replaces the existing page.

UPLOAD:
1. Download and extract the ZIP.
2. In the GitHub repository, open the existing awards.html.
3. Choose Edit or Delete/Upload.
4. Upload this new awards.html and commit directly to the main branch.
5. Wait about 1–3 minutes.
6. Open:
   https://x0SOULREAVER0x.github.io/vanguardfleet/awards.html
7. Refresh the page. On Android Chrome, use a new Incognito tab if an older cached copy appears.

This build:
- Preserves the existing Vanguard Fleet header, navigation, hero, top award display and footer.
- Removes the broken vanguard-awards-board.png reference.
- Replaces it with a searchable interactive official awards register.
- Uses the medal image files already present in the repository.
- Includes inline board JavaScript, so the register does not depend on a new external .js file.
