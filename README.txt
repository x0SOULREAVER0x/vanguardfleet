# Vanguard Fleet Management System — One-File Build

## Login
- Username: `admin`
- Password: `vanguard`

## Included
- Login screen
- Fleet Command dashboard
- Personnel management
- VPIN personnel jackets
- Application review
- Ships
- Academy
- Awards
- Qualifications
- Missions
- Fleet orders
- Settings and audit log
- Local browser storage
- JSON export

## Use
Upload `VFMS.html` to the GitHub repository and open it directly from GitHub Pages.

This is a local prototype. It does not yet provide secure real-world authentication or a shared online database.
